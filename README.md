# Caesar Cipher Python
 
A Caesar Cipher made in Python

## Functions: Encryption, Decryption

### Encryption: 
```
Type 'encode' to encrypt, type 'decode' to decrypt:
encode
Type your message:
hello
Type the shift number:
5
The encoded text is: mjqqt
```

### Decryption:
```
Type 'encode' to encrypt, type 'decode' to decrypt:
decode
Type your message:
mjqqt
Type the shift number:
5
The decoded text is: hello
```
